# 基础语法

元素Markdown语法
标题（Heading）# H1
## H2
### H3
粗体（Bold）**加粗文本** 或 __加粗文本__
斜体（Italic）*斜体文本* 或 _斜体文本_
引用块（Blockquote）> 引用块
有序列表（Ordered List）1. 项目1
2. 项目2
3. 项目3

1. 项目1
1. 项目2
1. 项目3
无序列表（Unordered List）- 项目1
- 项目2
- 项目3
代码（Code）`代码`
分隔线（Horizontal Rule）--- 或 *** 或 ___
链接（Link）[标题](https://www.example.com)
图片（Image）![图片alt](image.jpg)
