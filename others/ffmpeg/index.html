小于 1 分钟

FFmpeg

官网：A complete, cross-platform solution to record, convert and stream audio and video.

FFmpeg官网
FFmpeg官网帮助文档

通用命令

ffmpeg -i "输入文件.格式1" [执行参数] "输出文件名称.格式2"

常用参数：
-y              # 不询问，直接执行
-c:v copy       # 无损copy视频
-c:a copy       # 无损copy音频
-loglevel error # 输出日志级别

音频处理

-ab 320k # 设置比特率 320k
