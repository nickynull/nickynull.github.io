# 归档日志

# 查看归档日志

使用root登录后，切换到oracle用户，或者直接登录到oracle用户

su - oracle

rman target /

list archivelog all;

# 清理归档日志

提示

当归档日志满了之后，数据库将无法使用，需要清理归档日志。
如果数据库被关闭了，要先打开数据库再清理。
清理完之后如果有条件的话最好做一次备份。

使用root登录后，切换到oracle用户，或者直接登录到oracle用户

su - oracle

rman target /

清除指定日期的归档日志

delete archivelog all completed before 'sysdate-1';
