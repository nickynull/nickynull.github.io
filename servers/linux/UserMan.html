用户管理

Linux命令大全 | https://man.niaoge.com/

常用命令

# 用户管理
useradd     # 添加用户
userdel     # 删除用户
usermod     # 修改用户
passwd      # 用户密码管理

# 用户组管理
groupadd    # 添加用户组
groupdel    # 删除用户组
groupmod    # 修改用户组
newgrp      # 以相同的帐号，另一个群组名称，再次登入系统

# 批量处理
newusers    # 批量添加用户

常用文件

用户信息

/etc/passwd

内容格式：用户名:口令:用户标识号:组标识号:注释性描述:主目录:登录Shell

伪用户

bin 拥有可执行的用户命令文件 
sys 拥有系统文件 
adm 拥有帐户文件 
uucp UUCP使用 
lp lp或lpd子系统使用 
nobody NFS使用

用户登录信息

/etc/shadow

内容格式：登录名:加密口令:最后一次修改时间:最小时间间隔:最大时间间隔:警告时间:不活动时间:失效时间:标志

用户组信息

/etc/group

内容格式：组名:口令:组标识号:组内用户列表
