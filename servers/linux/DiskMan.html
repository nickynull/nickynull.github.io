磁盘管理

Linux命令大全 | https://man.niaoge.com/

df

df（英文全称：disk free）：列出文件系统的整体磁盘使用量

df -h   # 以KB以上的单位来显示

du

du（英文全称：disk used）：检查磁盘空间使用量

du -h /home     # 查看 /home 目录及其内容磁盘占用情况
du -hs /home    # 查看 /home 目录磁盘总占用情况
du -hs /home/*  # 查看 /home 目录下，第一子级内容磁盘总占用情况

fdisk

fdisk：用于磁盘分区

fdisk -l        # 查看硬盘情况

挂载硬盘

# 挂载新硬盘
lsblk           # 查看所有可用块设备的信息
blkid           # 查看块设备的文件系统类型
fdisk -l        # 查看硬盘情况
df -h           # 查看硬盘挂载情况
mkfs.ext4 /dev/sdb      # 格式化硬盘，其中/dev/vdb为硬盘路径
mkdir /mydata           # 创建将要挂载的目录
mount /dev/sdb /mydata  # 将硬盘/dev/vdb挂载到/mydata目录
# 开机启动后自动挂载
echo '/dev/sdb /mydata ext4 defaults 0 0' >> /etc/fstab
