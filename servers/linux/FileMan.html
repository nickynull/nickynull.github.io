# 常用文件管理命令

# ls

（英文全拼：list files）: 列出目录及文件名open in new window

ls -l

# cd

（英文全拼：change directory）：切换目录open in new window

cd /

# pwd

（英文全拼：print work directory）：显示目前的目录open in new window

# mkdir

（英文全拼：make directory）：创建一个新的目录open in new window

mkdir -p test/test/test/

# rmdir

（英文全拼：remove directory）：删除一个空的目录open in new window

rmdir -pv test/test/test/

# cp

（英文全拼：copy file）: 复制文件或目录open in new window

# rm

（英文全拼：remove）: 删除文件或目录open in new window

# mv

（英文全拼：move file）: 移动文件与目录，或修改文件与目录的名称open in new window
