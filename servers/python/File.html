# Python 文件操作

# open

answered by ChatGPT

  • 'r'：只读模式（默认模式）

    • 打开文件用于读取数据。
    • 文件指针位于文件的开头。
    • 如果文件不存在，会引发FileNotFoundError异常。

  • 'w'：写入模式

    • 打开文件用于写入数据。
    • 如果文件存在，会将其内容清空；如果文件不存在，则创建一个新文件。
    • 文件指针位于文件的开头。

  • 'x'：独占创建模式

    • 打开文件用于写入数据，但只能用于创建新文件。
    • 如果文件已经存在，则会引发FileExistsError异常。
    • 文件指针位于文件的开头。

  • 'a'：追加模式

    • 打开文件用于追加数据。
    • 如果文件存在，文件指针位于文件的末尾；如果文件不存在，则创建一个新文件。
    • 写入操作会在文件末尾添加数据，而不会覆盖原有数据。

  • 'b'：二进制模式

    • 打开文件用于处理二进制数据。
    • 可以与其他模式字符组合使用，如'rb'、'wb'等。

  • 't'：文本模式（默认模式）

    • 打开文件用于处理文本数据。
    • 可以省略，因为文本模式是默认模式。
    • 可以与其他模式字符组合使用，如'rt'、'wt'等。

  • '+'：更新模式

    • 打开文件用于同时进行读取和写入操作。
    • 文件指针位于文件的开头。
    • 可以与其他模式字符组合使用，如'r+'、'w+'、'a+'等。
    • 读取操作会从文件指针的位置开始读取数据。
    • 写入操作会在文件指针的位置写入数据，并且会覆盖原有数据。如果文件指针位于文件末尾，则写入操作会在文件末尾添加数据。
