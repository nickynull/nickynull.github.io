# Canvas

HTML5 的绘图画布

Canvas 接口文档open in new window
Canvas 参考手册 | 菜鸟教程open in new window

注意

Canvas 宽高需要通过属性指定，不可使用 CSS 设置，否则可能会出现不可预期的异常

<canvas width="800" height="600"></canvas>

# 基本概念

  • 画布：Canvas 对象
  • 画笔：通过 canvas.getContext() 获取，一个画布对应一个画笔，不可交叉使用
  • 坐标：原点(0,0)在左上角，x轴向右延申，y轴向下延申
  • 路径：绘图时线条的走向
  • 渐变：颜色的变化情况
    • createLinearGradient：创建线性渐变
    • createRadialGradient：创建径向渐变
const canvas = document.getElementById('canvas');
const ctx = canvas.getContext()

# beginPath

beginPath 方法用于开始一条路径，或重置当前的路径。

提示

绘图方法从路径的开始绘制到结束。
如果发现绘图效果与预期有出入，可以考虑检查 beginPath

# 绘制线条

ctx.beginPath()
ctx.moveTo(50, 50)  // 移动到绘图起点
ctx.lineTo(100, 25) // 从起点到目标点，画路径
ctx.lineTo(150, 50) // 从起点到目标点，画路径
ctx.stroke()        // 画线

# 绘制矩形

ctx.beginPath()
ctx.rect(0, 0, 50, 100) // 起点(0,0) 宽高(50,100)
ctx.stroke()            // 画线

ctx.beginPath()
ctx.rect(0, 0, 50, 100) // 起点(0,0) 宽高(50,100)
ctx.fill()              // 填充

ctx.beginPath()
ctx.fillRect(0, 0, 50, 100) // 起点(0,0) 宽高(50,100)，填充
